Graphices 是Python中的画图模块, 可以通过这个扩展模块来绘制各种简单的图形, Grahices 是基于 Python 内置的Tkinter模块扩展的一个绘图库！目前提供了9种绘图对象［Point, Line, Circle, Oval, Rectangle, Polygon, Text, Entry(text-based input), Image］! Graphices 的 GraphicsObject 实现了基础的方法, 如:draw(self, graphwin) 就是把当前对象绘制在Tkinter窗体上! Python 库种默认是没有提供Graphices模块的, 可以从这里下载Graphices.py
